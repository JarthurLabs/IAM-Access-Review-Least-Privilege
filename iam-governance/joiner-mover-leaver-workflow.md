# Joiner / Mover / Leaver Workflow

## Joiner

1. Manager submits access request.
2. Role is matched to RBAC matrix.
3. System owner approves access.
4. IT grants minimum necessary permissions.
5. MFA requirement is confirmed.
6. Evidence is recorded.

## Mover

1. Manager reports job role change.
2. Existing access is reviewed.
3. Access no longer needed is removed.
4. New access is granted by role.
5. Privileged access requires separate approval.

## Leaver

1. Manager or HR triggers offboarding.
2. Account access is disabled same day.
3. SaaS sessions are revoked.
4. Shared drive and group access is removed.
5. File ownership is transferred if needed.
6. Completion evidence is retained.
