# Privileged Account Approval Workflow

## Purpose

Privileged access is reviewed separately because admin permissions create higher business risk.

## Required Evidence

- User name
- System name
- Access level requested
- Business justification
- Manager approval
- System owner approval
- MFA confirmation
- Review date
- Expiration date for temporary access

## Approval Rule

Privileged access should be assigned to named users, protected by MFA, and reviewed at least quarterly.
