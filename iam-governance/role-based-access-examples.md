# Role-Based Access Examples

| Role | Approved Access |
|---|---|
| Billing Specialist | Billing app editor, customer file viewer |
| Finance Analyst | Billing app editor, finance reports |
| Sales Associate | CRM editor, customer contact records |
| Operations Manager | CRM admin, project management admin |
| IT Administrator | Admin console, security settings, user management |

## Least Privilege Rule

Access should be assigned based on the smallest permission set needed to complete the user’s job.
